## Using the purchased block
For using the block, you have to first [install the bb-cli](https://docs.appblocks.com/docs/Getting%20Started/Setting%20up%20the%20CLI).

### Paid block
As a logged in user, do the following steps from Manage Blocks.
* In the manage blocks table, switch to purchased blocks tab and find the block.  
![image](https://user-images.githubusercontent.com/33730398/217721029-cb899598-6837-47b3-9408-0fc3be1d93da.png)

* Install code by clicking the download button under the name of your block.
* Build with blocks

### Free purchased block
This can be done as a logged in or guest user.
* Find the free block of your liking and in the right hand side of the block detaild page, you will find a command that can be copied and pasted in bb cli.   
  ![image](https://user-images.githubusercontent.com/33730398/217720466-3e16a5e2-7d20-4b7c-a8b7-0c050858f900.png)
* Build with blocks.


## Deploying the app
Upload a blocks before publish each block with the following command  

     > bb upload -env dev    
     
The following command for deploying the app will prompt to choose uploaded blocks, and then the deploying method  

     > bb deploy

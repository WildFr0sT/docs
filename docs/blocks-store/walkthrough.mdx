# Blocks Store Walkthrough

### Blocks Store

The Blocks Store is a registry where developers can create and publish blocks or use blocks published by other developers. 
(Insert a general paragraph)

![image](https://user-images.githubusercontent.com/33730398/217148905-a4ceda71-114c-4049-b31a-b2b53d5c22fe.png)

You can find blocks that are published by Appblocks users here. Blocks can be free or paid, listed with seller's name and other necessary details. To make finding blocks easier,
* Search - You can search for different blocks that cater to your needs by entering keywords in the search bar, as shown in the above picture.
* Filter - You can filter the store for blocks by the following conditions: Appblocks version, Language/Frameworks, Block Type, Visibility, Depedencies, Category

### Visiting a Block

When you click any Block in the Blocks Store, you are redirected to the Block Details page. It gives you all details about the particular block. There are four tabs present in it, viz. ReadMe, Dependencies, Versions and Composable Block. If the Block you are visiting is an App Block, there will be a 5th tab called Structure.

![Blocks Store-details](https://user-images.githubusercontent.com/33730398/159855841-9b7f0cde-c43f-41ab-b943-51884454c280.png)

#### ReadMe
The ReadMe tab contains general information about this particular Block. To reuse a Block in your app, copy the link given below "Install" on the top right corner of this tab and _______. 

![Blocks Store-details-1](https://user-images.githubusercontent.com/33730398/159855910-e1f265b2-a853-43ea-8bd8-48ad3d77677e.png)
ReadMe also gives the number of downloads, tags, list of contributors and other such useful information about the Block.

#### Dependencies
In the Dependencies tab, as shown below, all dependent Blockes of the chosen Block is listed down. Click *here* to read more.

![Blocks Store-details-12](https://user-images.githubusercontent.com/33730398/159856319-ab155d86-4167-4b6e-b3f0-ad4f04f81146.png)

#### Versions
The Versions tab will display all versions of the Block till date. 
![Blocks Store-details-8](https://user-images.githubusercontent.com/33730398/159856147-af169e37-e8cd-4a34-99dc-55befa0b2d68.png)

#### Composible Block
All composible Blockes are listed in this tab as shown below. Click *here* to read more.
![Blocks Store-details-4](https://user-images.githubusercontent.com/33730398/159856037-06dc32c0-8d71-47a3-9e10-a7683204abf9.png)

#### Structure
If the visited Block is an App Block, the Structure tab will be present. It will display the entire app structure. 

As shown below, the flow diagrams give you a good understanding of how the UI Blockes are connected to function Blockes and the function interfaces present in the app Block.

![Structure](https://user-images.githubusercontent.com/33730398/159856612-c69f9228-b470-4b97-9687-49f9c637e41d.png)

Another salient feature is that testing can be performed using the test endpoints.
